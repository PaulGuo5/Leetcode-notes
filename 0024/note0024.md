# Leetcode [24. Swap Nodes in Pairs](https://leetcode.com/problems/swap-nodes-in-pairs/)
- Type: Linklist (iterative/recursive)
- Approach:
	1. Iterative:
		- Set a current pointer to store previous list node
	2. Recursive:
		- Duplicated operation for each reverse
- Complexity:
	- Time: O(n)
	- Space: O(1)
